# cadA — Protótipo de Loja de Jogos (Flask)

Projeto inicial em Python/Flask criado como base para uma loja de jogos semelhante à Steam.

Instruções rápidas (Windows):

1. Criar e ativar ambiente virtual

```powershell
python -m venv venv
venv\Scripts\Activate.ps1
```

2. Instalar dependências

```powershell
pip install -r requirements.txt
```

3. Rodar o servidor

```powershell
python app.py
```

O frontend estará disponível em `http://localhost:5000`.

Próximos passos sugeridos:
- Integrar Steam OpenID para login
- Consumir a Steam Web API para buscar lista real de jogos
- Criar páginas de detalhes, wishlist e sistema de avaliações
